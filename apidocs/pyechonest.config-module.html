Variables Details [hide private]

MP3_BITRATE

Default bitrate for MP3 output. Conventionally an integer divisible by 32kbits/sec.
Value:
192

OBEY_RATE_LIMIT

The Echo Nest limits users to 120 api calls per minute. By default, pyechonest enforces this limit locally. Set this variable to False to turn of local enforcement. The Echo Nest api will still throttle you.
Value:
True
