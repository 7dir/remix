Class HTMLNode

source code

HTML 4 compliant Node

Node renders as standard HTML 4 markup.

>>> br = HTMLNode("br")
>>> str(br)
'<br>'
>>> img = HTMLNode("img", src="http://domain.com/image.jpg")
>>> str(img)
'<img src="http://domain.com/image.jpg">'
>>> p = HTMLNode("p")
>>> p.append(img)
<StupidXML.HTMLNode: "img">
>>> str(p)
'<p><img src="http://domain.com/image.jpg"></p>'
