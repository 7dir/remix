Remix

Echo Nest Remix is the Internet Synthesizer. Make amazing things from music, automatically.
Turn any music or video into Python, Flash, or Javascript code.

What:

Echo Nest Remix is the Internet Synthesizer. It lets you make amazing things with music and video, automatically and algorithmically.

Remix has done the following: rearranged the syllables of a vocal line, walkenized and cowbellized hundreds of thousands of songs in a week, reversed basically everything, beat matched two songs, split apart DJ mixes by their individual tracks, made new kinds of video mashups, corrected sloppy drumming, synced video to a song, transitioned between multiple covers of the same song, made a cat play piano, and made it swing.

We're making this available as an open source SDK for Python and JavaScript that can be easily installed on Mac OS X, Linux, or Windows. It accesses the Echo Nest Analyze API at http://developer.echonest.com and lets you play with the results to make new concoctions of audio and video. You'll need an Echo Nest API key to use Remix: instructions are below.

Step 2: Set up your Echo Nest API Key

To use Remix, you need to get API key from the Echo Nest Developer Center, by registering an account.

Once you've got your key, read this key install tutorial for Windows, Mac or Linux.

For the experts in the room, just set the environment variable you need: 

export ECHO_NEST_API_KEY="your api key"

Step 3: Try an example

Once you've got your API key set up, try one of the examples.

  • On Windows they're installed to your Python folder (probably C:\Python27\echo-nest-remix-examples\.)
  • On Mac they are in /usr/local/share/echo-nest-remix-examples.
  • On Linux, they are also in/usr/local/share/echo-nest-remix-examples!

A good place to start is with one.py. Turn on your terminal, go to the examples folder, and 

python one/one.py music/Karl_Blau-Gnos_Levohs.mp3 AllOnTheOne.mp3
Congratulations! You've made your first remix with Remix!

Step 4: Make something beautiful

For inspiration on what to do with remix, you can check the examples below. You can also take a look at Adam Lindsay's tutorial, or you can check out the full API documentation.

Finally, here's a bit of code to whet your appetite. Happy remixing! 

"""Reverse a song by playing its beats forward starting from the end of the song"""
import echonest.audio as audio

# Easy around wrapper mp3 decoding and Echo Nest analysis
audio_file = audio.LocalAudioFile("NeverGonnaTellIt.mp3")

# You can manipulate the beats in a song as a native python list
beats = audio_file.analysis.beats
beats.reverse()

# And render the list as a new audio file!
audio.getpieces(audio_file, beats).encode("NeverGonnaTellItBackwardsByBeat.mp3")
